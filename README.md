# Pharmacy Management System - Practice Version

This is a **Practice Version** of a Pharmacy Management System built with **HTML, CSS, and JavaScript**.  
It simulates a basic pharmacy system for managing medicines, sales/billing, and generating invoices.  

This project is designed for learning purposes and as a foundation for building a **real Django-based system**.

---

## **Features**

### 1. Login System (Front-End)
- Simple login page with username & password
- Redirects to dashboard on correct credentials
- Front-end validation

### 2. Dashboard
- Summary cards showing:
  - Total Medicines
  - Total Stock Quantity
  - Total Sales Items
- Updates live as you add or delete data

### 3. Medicines Module
- Add and delete medicines
- Table displays medicine name and quantity
- Dashboard cards update dynamically

### 4. Sales / Billing Module
- Add items to bill with quantity and rate
- Auto-calculation of amount per item and total bill
- Delete items from bill
- Live update of dashboard cards

### 5. Invoice Generation
- Generate printable invoice from sales table
- Includes date, items, quantities, rates, amounts, total
- Opens in new browser window ready to print

---

## **Technologies Used**
- **HTML** – Structure of pages and forms
- **CSS** – Styling, layout, and dashboard cards
- **JavaScript** – Interactivity, dynamic tables, total calculation, invoice generation
